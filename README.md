# 📊 Advanced Sales Analytics Dashboard

An End-to-End Data Analytics Project built using **Excel, SQL, Python, and Power BI** to analyze Superstore sales data and generate actionable business insights through an interactive dashboard.

---

## 🚀 Project Overview

This project demonstrates a complete data analytics workflow:

- Data Cleaning & Feature Engineering (Excel)
- Business Analysis using SQL
- Exploratory Data Analysis (Python)
- Interactive Dashboard (Power BI)

The objective is to transform raw sales data into meaningful business insights that support data-driven decision-making.

## 🔍 Key Business Insights

- Technology generated the highest sales.
- Consumer segment contributed the highest revenue.
- West region achieved the best sales performance.
- Higher discounts negatively impacted profit.
- Standard Class was the most preferred shipping mode.
